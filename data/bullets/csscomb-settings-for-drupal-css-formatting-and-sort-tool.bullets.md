- Use CSScomb tool to format and sort CSS properties in css, scss, sass or less files.
- Follow the recommendations in the CSS formatting guidelines for the used values.
- Base the sort order of CSS properties on the Yandex configuration file and improve for SASS/LESS by putting variables first.
- Copy the configuration into the CSScomb package settings file under Preferences in Sublime Text after installation of the plugin.
- For most other use cases, put the configuration in a file named `.csscomb.json` in your local home folder to make it applicable to all your projects.
- Use csscombx, a fork of Csscomb tool specifically intended for Drupal, as an alternative.
- Install nodejs on your system to use npm, a package manager for many popular javascript projects.
- Install csscombx globally.
- After installing, cd into your theme's root folder or any folder containing style files to be formatted and make a dry run without changing anything.
- Use the -v option on the command for verbose and -l for lint.
- Customize formatting settings by copying the content of the drupal.json configuration file and pasting into .csscombx.json file in the root of your theme, a particular style folder, webroot folder or any other folder.
- Automate the formatting task and run it whenever the style file(s) are saved after being edited using the gulp-csscombx plugin.
- Install gulp-cli globally.
- Create package.json file in the root of your theme, module or any other project and put the provided content into it.
- Run the provided command after creating the package.json file.
- Create a gulpfile.js file in the root of your theme, module or any other project and put the provided content into it.
- Adjust mySrcStyles variable's value for the content you want to be formatted and run the provided command.
- Terminate the gulp watcher's task by pressing Ctrl+C keyboard shortcut when you are done with developing your styles.
- If the drupal.json default configuration file's settings do not comply with Drupal CSS coding standards, create a new issue for the CSS component of the Drupal core, post a comment on the page, create the issue or make a pull request on the csscombx.
- If there are difficulties with the gulp-csscombx plugin, post the issue on its GitHub repository.
